HTML Notes

HTML (HyperText Markup Language) is the standard language for creating web pages and web applications.

Below you can see an example of a boilerplate HTML document that includes various elements. HTML elements typically have an opening and closing tag with content in between.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML 5 Boilerplate</title>
    <link rel="stylesheet" href="style.css">
    <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/5.1.3/css/bootstrap.min.css">
    <link rel="icon" href="images/favicon">
    <meta property="og:title" content="HTML 5 Boilerplate">
    <meta property="og:type" content="website">
    <meta property="og:url" content="http://example.com">
    <meta property="og:image" content="http://example.com/image.jpg">
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
    <script src="index.js"></script>
  </body>
</html>

Common HTML Elements

  • <html>: The root element of an HTML page
  • <head>: Contains meta-information about the document, such as the title, meta tags, and links to stylesheets
  • <title>: Sets the title of the document, which is displayed in the browser’s title bar or tab
  • <body>: Contains the content of the document, such as text, images, links, etc.
  • <div>: A division of an HTML document
  • <p>: A paragraph element; browsers automatically add a single blank line before and after each <p>
  • <br>: A single line break
  • <a>: A hyperlink; the href attribute specifies the URL of the page the link goes to
  • <!-- -->: A comment element (text can be added in the middle)

Semantic HTML

  • <header>: Defines a header for a document or section
  • <nav>: Defines a set of navigation links
  • <section>: Defines a section in a document
  • <article>: Defines independent, self-contained content
  • <footer>: Defines a footer for a document or section
  • <aside>: Defines content aside from the main content
  • <details>: Defines additional details that the user can view or hide
  • <figcaption>: Defines a caption for a <figure> element
  • <figure>: Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
  • <main>: Specifies the main content of a document
  • <mark>: Defines highlighted text
  • <summary>: Defines a visible heading for a <details> element
  • <time>: Defines a specific time (or datetime)

HTML Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like name="value".

  • class: Specifies one or more class names for an element (used for CSS and JavaScript).
  • id: Specifies a unique id for an element (used for CSS and JavaScript).
  • style: Specifies an inline CSS style for an element.
  • title: Specifies extra information about an element (displayed as a tooltip).
  • alt: Specifies an alternative text for images, if the image cannot be displayed.

HTML Canvas

The HTML <canvas> element is used to draw graphics on a web page via JavaScript. Below is an example of how to use the <canvas> element.

<canvas id="myCanvas" width="400" height="200" style="border:1px solid #ffffff;"></canvas>
<script>
// Get the canvas element by its ID
var canvas = document.getElementById('myCanvas');
// Get the 2D drawing context
var ctx = canvas.getContext('2d');

// Set the fill style to a color
ctx.fillStyle = '#00BEEF';
// Draw a filled rectangle
ctx.fillRect(10, 10, 150, 100);

// Set the stroke style to a color
ctx.strokeStyle = '#FFFFFF';
// Draw a rectangle outline
ctx.strokeRect(200, 10, 150, 100);

// Set the font style
ctx.font = '20px Arial';
// Draw text on the canvas
ctx.fillText('Hello Canvas', 50, 180);
</script>

HTML Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like name="value". Some common attributes are listed below. Each of these attributes (except for alt) are global, meaning they can be used universally with all HTML elements.

  • class: Specifies one or more class names for an element (used for CSS and JavaScript).
  • id: Specifies a unique id for an element (used for CSS and JavaScript).
  • style: Specifies an inline CSS style for an element.
  • title: Specifies extra information about an element (displayed as a tooltip).
  • alt: Specifies an alternative text for images, if the image cannot be displayed.

Note that a range of attributes can be used for different elements. For example, if you'd like to add styling to an element without using a class or id, you can use the style tag. 

<p style="color:red;">This is a red paragraph.</p>

CSS Notes

CSS (Cascading Style Sheets) is used to style and layout web pages. Below are some common concepts and properties.

Selectors

  • Selectors: Used to select the HTML element(s) you want to style.
  • Element Selector: Selects all elements of a given type (e.g., p).
  • Class Selector: Selects all elements with a given class (e.g., .classname).
  • ID Selector: Selects a single element with a given ID (e.g., #idname).
  • Attribute Selector: Selects elements with a specific attribute (e.g., [type="text"]).

Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree and consists of: margins, borders, padding, and the actual content.

  • Box Model: Describes the rectangular boxes generated for elements in the document tree.
  • Margin: The outermost layer, used to create space around elements.
  • Padding: The space between the content and the border of an element.
  • Border: The line surrounding the padding (if any) and content.

Common Properties

  • background-color: Sets the background color of an element.
  • font-family: Specifies the font for an element.
  • text-align: Specifies the horizontal alignment of text.
  • position: Specifies the type of positioning method used for an element (static, relative, absolute, fixed, or sticky).
  • width: Sets the width of an element.
  • height: Sets the height of an element.

Flexbox

CSS Flexbox is a layout model that allows for the easy alignment of elements within a container.

  • Flexbox: A layout model for designing complex layouts with ease.
  • flex: A shorthand property for flex-grow, flex-shrink, and flex-basis.
  • justify-content: Aligns the flexible container’s items when the items do not use all available space on the main-axis.
  • align-items: Aligns items along the cross-axis.

Grid Layout

CSS Grid Layout is a two-dimensional layout system for the web.

  • Grid Layout: A two-dimensional layout system for web design.
  • grid: A shorthand property for grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns, and grid-auto-flow.
  • grid-template-columns: Defines the columns of the grid with a space-separated list of values.
  • grid-template-rows: Defines the rows of the grid with a space-separated list of values.

Media Queries

Media queries are used to apply different styles for different devices or screen sizes.

  • Media Queries: Used to apply different styles for different devices or screen sizes.
  • @media: A rule used to apply styles based on the result of one or more media queries.
